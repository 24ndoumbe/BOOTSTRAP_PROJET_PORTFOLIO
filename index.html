MES PROJETS

Application de Gestion de Frais














Création d'une application de gestion de frais...

Cette application permet la gestion de frais professionnels avec une interface dédiée aux visiteurs et aux comptables. L'utilisateur peut enregistrer ses frais, les modifier, les consulter, et les envoyer pour validation. L'application est basée sur une architecture sécurisée avec authentification. Si l'utilisateur est un visiteur, il pourra remplir des fiches de remboursement contenant le nombre de nuitées, de repas, de kilomètres, les frais exceptionnels qu'il a effectués avant le 10 de chaque mois ainsi que consulter les fiches déjà remplis. S'il est comptable, il pourra analyser les fiches de remboursement des visiteurs afin de les valider ou les refuser.

Jeu 2048










jeu de puzzle

Le projet consiste à développer une version personnalisée du célèbre jeu "2048". Ce jeu de puzzle repose sur la combinaison de tuiles ayant les mêmes valeurs pour atteindre la valeur cible de 2048. L'objectif de ce projet est de créer une version améliorée avec des variantes sur le système de calcul, la grille, et des options permettant de personnaliser l’expérience du joueur.
